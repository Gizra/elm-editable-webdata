EditableWebData
========

> An EditableWebData represents an Editable value, along with WebData.


## Installation

`elm-package install Gizra/elm-editable-webdata`

## Tests

Install packages: `npm install -g elm-test elm-verify-examples`

Execute tests: `./execute-tests`
